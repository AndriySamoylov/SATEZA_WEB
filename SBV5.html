Explore SABA INFO V.1.5

Overview

SabaInfo 1.5 is a standalone application developed in Python, representing the first version released for public use. The application was created to provide structured and accessible information about “Saba,” allowing users to explore relevant data efficiently. This version of SabaInfo was designed with clarity and usability in mind, offering an intuitive interface that supports straightforward navigation and interaction. By making the application publicly available, the development team aimed to provide a practical tool that demonstrates reliable data handling and information presentation.

Background

Main Idea : The idea for Sabalnfo originated in the summer of 2024 as a personal learning project to improve Python skills. Inspired by our friend Saba, who frequently shared interesting and unusual statements in a humorous yet serious manner, we decided to create a system that could organize and present these insights. The concept was to develop an application in which users could query specific information about Saba, such as his age or other notable details. Initially envisioned as a mobile application, the project evolved into a more structured system capable of searching and displaying information efficiently. This early idea laid the foundation for Sabalnfo, combining the goals of technical learning with a structured approach to presenting user-friendly and queryable information.

Development : The development of SabaInfo began shortly after the summer of 2024 with the release of version 1.1. This initial version was a minimal Python program of approximately 100 lines of code. It lacked a user interface and functioned solely through text input, requiring precise queries to retrieve information. Users could ask a few predefined questions and receive corresponding answers, but no flexible search capabilities were available.

Version 1.2 introduced a more advanced search mechanism. Users could input queries in lowercase or in various phrasing formats, such as “how old is Saba” or “how many years is Saba old,” and the system would interpret the intent and provide the correct response. While still lacking a graphical interface, this version significantly improved usability and query flexibility.

With version 1.3, the application exceeded 100 lines of code and implemented basic user account functionality. Users were required to create an account, agree to confidentiality terms, and log in with a password, ensuring controlled access to the information within the app.

Version 1.4 marked the first iteration with a graphical user interface, developed using Python’s Tkinter library. This version introduced an interface closely resembling the final 1.5 design, while retaining key differences. It served as a demo version of 1.5, providing the first experience of a fully interactive application, laying the groundwork for the public release.
